# UI/UX Specification - HoSo Reader

## Concept: E-ink & Print First
- **Theme:** High Contrast (Black & White). Không dùng gradient hay animation phức tạp.
- **Typography:** Ưu tiên Font Serif (có chân) cho nội dung dài để dễ đọc trên Nova Air C.
- **Layout:** - Chế độ "Web View": Tràn màn hình, menu tinh gọn.
  - Chế độ "Print View": Tự động ngắt trang (A4), có đánh số trang, header/footer chuyên nghiệp.
- **Nova Air C Opt:** Nút bấm to, rõ ràng, hạn chế cuộn (scroll) - ưu tiên chuyển trang (pagination).

## Components:
- **File Dropper:** Vùng kéo thả file .md hoặc .doc.
- **Project Navigator:** Danh sách các folder dự án (Slavii, Web Vita...).
- **Export Toolbar:** Nút "Print to PDF", "Share Link", "Save to Project Folder".
