# Master Prompt: HoSo WebApp Builder

"Build a web application that converts Markdown and .docx files into professional, print-ready documents. 
The app is located at D:\hoso.

**Core Requirements:**
1. **Nova Air C Optimization:** The UI must be high-contrast (Black/White), no shadows, no complex transitions. Use large touch targets.
2. **Reading Experience:** Support a 'Book Mode' with pagination instead of long scrolling.
3. **Printing:** Implement a CSS-based print engine that ensures professional A4 layouts with page numbers.
4. **Processing:** - Drag & drop .md or .docx. 
   - Generate a live preview that looks exactly like the printed version.
   - Option to export as a standalone HTML file.
5. **Context:** Read ui.md, implement.md, and task.md for further details. 

Let's start by creating a Vite + React + Tailwind project with a high-contrast layout."
